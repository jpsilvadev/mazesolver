# mazesolver

A DFS maze solver implemented in Python with Tkinter.

## Prerequisites
- **Ensure you have Python installed**
```bash
$ python3 --version
```

## Getting Started
1. Clone the repo:
```bash
$ git clone https://github.com/jpsilvadev/mazesolver.git
$ cd mazesolver
```

2. Run the program:
```bash
$ python3 main.py
```
> Note: You might need to install `python3-tk` on deb-based systems.
